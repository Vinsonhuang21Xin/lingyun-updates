# 灵云项目助理 - 版本更新仓库

这个仓库用于托管灵云项目助理APP的版本更新文件。

## 文件说明

- `version.json` - 版本配置文件，包含最新版本信息和下载地址
- `lingyun-v{版本号}.wgt` - APP更新包（热更新）

## 如何发布新版本

1. 修改 `version.json` 文件：
   - 更新 `version` 为新版本号
   - 更新 `downloadUrl` 为新的下载地址
   - 更新 `updateLog` 为新版本的更新日志

2. 上传新的更新包文件（.wgt）

3. 提交到GitHub
